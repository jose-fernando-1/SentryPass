# SentryPass
Password manager

## Requirements
- Go 1.24.1

## Running

1. **Clone the repository**:
   ```sh
   git clone https://github.com/jose-fernando-1/SentryPass.git
   ```

    ```sh
   cd SentryPass
    ```

2. **Install dependencies**:
   ```sh
   go mod tidy
   ```

3. **Compile and execute**:
    ```sh
   go run main.go
    ```
   **Or**:
    ```sh
    go build -o sentrypass
    ```
    ```sh
    .\sentrypass.exe
    ```
